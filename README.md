# Korea Czech Bet

Static match prediction page for a Korea vs Czech football bet.

## Local Preview

```bash
python3 -m http.server 4173 --bind 127.0.0.1 --directory outputs/soccer-bet
```

Open `http://127.0.0.1:4173`.

## Render

This repo includes `render.yaml` for a Render Static Site.

- Runtime: Static
- Build command: `true`
- Publish directory: `./outputs/soccer-bet`
